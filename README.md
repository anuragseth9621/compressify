# Compressify

A smart image compressor built using Node.js, Express.js, Multer and Sharp.

## Features

* Upload Images
* Drag & Drop Upload
* Compression Quality Slider
* Dark Mode
* Mobile Responsive
* Download Compressed Images

## Live Website

https://compressify-dfiw.onrender.com/

## Tech Stack

* HTML
* CSS
* JavaScript
* Node.js
* Express.js
* Multer
* Sharp
* Render
