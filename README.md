# Compressify

Compressify is a free online image compression tool built using Node.js, Express.js, Multer, and Sharp. It allows users to upload images, reduce file size, preview results, and download optimized images instantly.

## 🚀 Live Demo

https://compressify-dfiw.onrender.com/

## ✨ Features

* Upload images from your device
* Drag and drop image upload
* Adjustable compression quality
* Image preview before compression
* Compare original and compressed sizes
* Download compressed images
* Dark mode support
* Fast image processing using Sharp
* Mobile responsive design

## 📦 Installation

```bash
git clone https://github.com/anuragseth9621/compressify.git
cd compressify
npm install
npm start
```

## 🎯 Usage

1. Open the application.
2. Upload an image.
3. Adjust compression quality.
4. Click **Compress Image**.
5. Download the optimized image.
